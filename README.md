# Investment Forecast Web App – Quantity + Unit Price Edition

Тази версия позволява да променяш:
- количество
- цена за единица (грам / дял / SOL)

Приложението автоматично смята:
- текуща стойност = количество × цена за единица
- 1 месец
- Q1 / Q2 / Q3 / Q4
- 1 / 2 / 3 / 4 години
